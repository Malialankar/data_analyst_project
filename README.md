# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using transactional data (3,900+ records) to uncover insights into spending patterns, customer segmentation, and product performance. The goal is to support data-driven business decisions.

---

## Dataset Summary
- Total Records: 3,900+
- Features: 18 columns
- Includes:
  - Customer demographics (Age, Gender, Location)
  - Purchase details (Item, Category, Amount, Season)
  - Behavior data (Discounts, Frequency, Ratings, Shipping)

---

## Tools & Technologies
- Python (Pandas)
- SQL (MYSQL)
- Power BI
- Excel

---

## Project Workflow

### 1. Data Cleaning & Preprocessing (Python)
- Loaded dataset using Pandas
- Handled missing values in review ratings using median
- Renamed columns for consistency
- Created new features:
  - Age Groups
  - Purchase Frequency
- Removed redundant columns

---

### 2. Data Analysis (SQL)
Performed analysis to answer key business questions:

- Revenue by gender and age group
- Customer segmentation (New, Returning, Loyal)
- Top-performing products and categories
- Impact of discounts on revenue
- Subscription vs non-subscription behavior
- Repeat customer analysis

---

### 3. Data Visualization (Power BI)
- Built interactive dashboard
- Visualized:
  - Revenue trends
  - Customer segments
  - Product performance
  - Discount impact

---

## Key Insights
- Identified high-value customers and loyal segments
- Found top-performing products based on ratings and sales
- Analyzed impact of discounts on customer behavior
- Observed differences in spending patterns by age group
